# 🏠 Real Estate Chatbot

An AI-powered Real Estate Assistant that helps users search, compare, and explore properties using natural language queries.

## Features

- Property Search
- Property Comparison
- AI Chat Assistant
- Location-Based Recommendations
- Real Estate Dataset Integration
- Responsive Frontend UI
- Gemini AI Integration

## Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/real-estate-chatbot.git
cd real-estate-chatbot
```

### Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
GEMINI_API_KEY=your_api_key_here
PORT=5000
```

Run Backend

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
